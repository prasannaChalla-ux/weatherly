Weatherly is a simple and lightweight weather application.
It displays current weather information for a selected city.
The app shows temperature, humidity, and wind speed.
It uses a weather API to fetch real-time data.
The interface is clean, minimal, and user-friendly.
This project is built only using frontend technologies.

Features

Displays current temperature

Shows humidity level

Shows wind speed

Weather condition icons

Simple and responsive UI
